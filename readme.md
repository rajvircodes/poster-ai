Developed during our college workshop, PosterAI is a web application that transforms user inputs into a professional magazine-style poster. By combining a personal photo, name, and description, the app uses AI to generate a high-quality layout.

🚀 Features
Custom Inputs: Users provide their name and a personal bio.

Photo Upload: Supports direct image uploads for the poster's main visual.

AI Styling: Automatically generates a magazine-grade layout and background.

Instant Export: Download the final poster as an image file.
